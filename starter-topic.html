About NyanBomb

NyanBomb is a video game that begun as a "NSI" project at Saint-Aubin La Salle It started as a fireworks simulation with physics and is now a playable and simple video game.

Technical details

Language: Python 3.9.1
External libraries:

  • Pygame 2.0.1 - Rendering and input

  • Nuitka 1.8 - Compiling binary

Source code: Github - Under CC0-1.0 license
